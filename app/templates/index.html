About the Model

Linear Support Vector Machine (Linear SVM)

This spam classifier uses a Linear Support Vector Machine (Linear SVM), a supervised machine learning algorithm that learns an optimal decision boundary (hyperplane) to separate spam and non-spam messages with maximum margin.

Feature Extraction

The model analyzes textual patterns using TF-IDF vectorization, including:

  • Presence of spam-indicative keywords (free, win, urgent, offer, etc.)
  • Term frequency and inverse document frequency of words
  • Message length and word distribution
  • Capitalization and punctuation patterns
  • Numerical and URL-related tokens

Training Data

The model was trained on the SMS Spam Collection dataset, containing both spam and legitimate messages. Linear SVM demonstrated strong generalization performance, achieving high F1-score and ROC-AUC during stratified cross-validation.
